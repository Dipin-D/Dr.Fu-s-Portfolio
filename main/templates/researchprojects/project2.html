AI-Enhanced Programming Skills at HBCUs

Funded by National Science Foundation

Explore Project ↓

Project Description

This project, led by Prairie View A&M University and Alabama A&M University, enhances programming education for engineering students at HBCUs via AI-driven Personalized Adaptive Learning (PAL) tools.

Smart Online Modules

For assignments, projects, and exams.

PAL Path Recommender

Powered by deep reinforcement learning.

Smart Programming Assistant

Using large language models.

Intellectual Merit

This research integrates reinforcement learning, RNNs, and LLMs into an adaptive education ecosystem, addressing persistent programming education challenges and pioneering new AI-based pedagogical methods.

Broader Impacts

Expanding HBCU research capacity and engaging underrepresented students in AI, this project strengthens diversity in STEM and prepares a future-ready workforce. Tools will be integrated into curricula and openly shared with the wider academic community.
